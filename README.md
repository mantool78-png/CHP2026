# Прогнозы ЧМ-2026

MVP сайта конкурса прогнозов к Чемпионату мира по футболу 2026. Проект рассчитан на обычный PHP/MySQL-хостинг Beget Start: без Composer, Node.js, Docker и постоянных фоновых процессов.

## Возможности

- Регистрация и вход участников.
- Ручное подтверждение взноса админом.
- Личный кабинет с прогнозами на матчи.
- Закрытие приема прогнозов за 5 минут до начала матча.
- Открытие прогнозов после старта матча.
- Ввод результатов матчей админом.
- Автоматический подсчет очков: 1 за исход, 3 за точный счет, 10 за чемпиона.
- Таблица лидеров и расчет текущего призового фонда.

## Установка локально или на Beget

1. Создайте MySQL-базу данных.
2. Импортируйте `database/schema.sql` через phpMyAdmin.
3. Скопируйте `config/config.example.php` в `config/config.php`.
4. Укажите реальные доступы к базе в `config/config.php`.
5. Загрузите файлы проекта на хостинг.
6. Убедитесь, что Apache использует `.htaccess`. Если домен можно направить прямо в папку `public`, это предпочтительный вариант.

Файл `config/config.php` содержит пароль базы и не должен попадать в GitHub.

## Доступ админа

После импорта схемы создается администратор:

- Email: `admin@example.com`
- Пароль: `password`

Это только тестовый пароль для первого входа. Сразу замените email и пароль администратора перед публичным запуском.

## Работа с матчами

В MVP расписание загружается вручную через админку. Это надежнее, чем парсить страницу FIFA в реальном времени. Когда официальный календарь будет финализирован, матчи можно добавить через админку или расширить проект CSV-импортом.

Официальный календарь: https://www.fifa.com/en/tournaments/mens/worldcup/canadamexicousa2026/articles/match-schedule-fixtures-results-teams-stadiums

## Что добавить перед реальным запуском

- Настройку пропорций выплат топ-10.
- Резервное копирование базы перед каждым игровым днем.
- Ограничение прогноза на чемпиона до старта турнира.
